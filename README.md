# Calculadora-py
Este projeto foi desenvolvido como atividade prática do curso de Análise de Dados da EBAC.

## Como executar o script:
 - Abra o terminal Linux (Ubuntu);
 - Vá até a pasta do arquivo;
 - Dê permissão de execução, caso necessário: `chmod 744 calculadora.sh`
 - Execute o arquivo: `./calculadora.sh`

# Explicação
O arquivo recebe dos usuários os dados/números a serem utilizados em equações básicas, como, soma, subtração, multiplicação, potenciação e divisão. Ele realiza essa função de forma simples e rápida.
